Performs a V-cycle multigrid method for solving a linear system.

Implements a V-cycle for solving a linear system using multigrid methods. The method involves smoothing, restriction, exact solve on the coarsest grid, prolongation, and correction.

Parameters
uArray of solution vectors for each grid level.
fArray of right-hand side vectors for each grid level.
N_startNumber of internal points in the finest grid.
levelsNumber of multigrid levels.
vNumber of smoothing iterations at each level.
dimDimension of the problem (1 or 2).
use_stencil9Flag for enabling 9 point stencil in the 2d case (0 or 1).
debugFlag for enabling debug output (1 to enable, 0 to disable).
Note
Arrays u and f should be pre-allocated for each level.
