Solves a linear system using the multigrid method.

Solves the linear system by performing multiple V-cycles until convergence or maximum iterations are reached.

Parameters
uThe solution vectors for each level of the grid hierarchy.
fThe right-hand side vectors for each level of the grid hierarchy.
NThe number of internal points in the finest grid.
levelsThe number of levels in the multigrid hierarchy.
vThe number of pre- and post-smoothing steps.
dimThe dimension of the problem (1 or 2).
use_stencil9Flag for enabling 9 point stencil in the 2d case (0 or 1).
print_errorOption to print the error after each cycle (0 or 1).
