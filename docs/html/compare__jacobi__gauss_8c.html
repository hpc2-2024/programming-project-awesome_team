Compare the runtime of the Multi-Grid method when using the Jacobi smoother against the Gauss-Seidel smoother.

Note
Usage: ./mg <dimension> <gridsize N> <levels> <smoothing steps> <iter>
Example: ./mg 2 19 2 2 10
Note: The dimension must be 1 or 2.
Optional flags:
-fopenmp: use this flag for shared memory parallelization, run "export OMP_NUM_THREADS=...\" before using to set the number of threads on your machine.
-fcycle: with this flag the multigrid method uses fcycle instead of vcycle
-stencil9: uses the 9-point stencil, only works if dimension=2
